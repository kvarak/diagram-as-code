# diagram-as-code

Trying to codify diagrams, based on [https://github.com/mingrammer/diagrams](https://github.com/mingrammer/diagrams)

## Getting started

Just follow the above link, but basically.

You need:
- Python 3.6+
- Graphviz
- pip install diagrams

And then run:
```
> python diagram.py
```
